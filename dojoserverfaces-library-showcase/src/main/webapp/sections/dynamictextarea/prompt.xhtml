A simple multi-line text input area. Its rows will grow automatically
required Indicates a value is required to be submitted for this component.
disabled Whether or not this component is disabled.
maxLength The maximum length of input to be allowed by the client.
trim Indication of whether or not leading and trailing whitespace are trimmed from the input field on the client.
cols The width of the text area.
selectOnClick Indication that all text should be selected when field receives focus with mouse
readOnly Whether or not this component is read-only.
