An component that allows one to select a value with a horizontally draggable handle.
showButtons Flag controlling the showing of the increment/decrement buttons at the ends of the slider. Default is true.
minimum The minimum value the slider can be set to. Default is 0
maximum The maximum value the slider can be set to. Default is 100.
discreteValues If specified, indicates that the slider handle has only 'discreteValues' possible positions, and that after dragging the handle, it will snap to the nearest possible position. Thus, the slider has only 'discreteValues' possible values. For example, if minimum=10, maximum=30, and discreteValues=3, then the slider handle has three possible positions, representing values 10, 20, or 30. If discreteValues is not specified or if it's value is higher than the number of pixels in the slider bar, then the slider handle can be moved freely, and the slider's value will be computed/reported based on pixel position (in this case it will likely be fractional, such as 123.456789).
pageIncrement If discreteValues is also specified, this indicates the amount of clicks (that is snap positions) that the slider handle is moved via page up/page down keys. If discreteValues is not specified, it indicates the number of pixels. Default is 2
clickSelect Flag to indicate if clicking the slider bar changes the value or not.
slideDuration The time in ms to take to animate the slider handle from 0% to 100%, when clicking the slider bar to make the handle move.
