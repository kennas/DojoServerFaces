Vertical Rule are Hash marks for Vertical Slider
count Number of hash marks to generate. Default is 3.
container For verticalSlider, this is either "rightDecoration" or "leftDecoration", and indicates whether this rule goes above or below the slider.
labels Array of text labels to render - evenly spaced from left-to-right or bottom-to-top. Alternately, minimum and maximum can be specified, to get numeric labels.
numericMagin Number of generated numeric labels that should be rendered as '' on the ends when labels[] are not specified. Default is 0.
minimum Leftmost label value for generated numeric labels when labels[] are not specified. Default is 0.
maximum Rightmost label value for generated numeric labels when labels[] are not specified. Default is 1.
